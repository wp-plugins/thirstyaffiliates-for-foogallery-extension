Extension for FooGallery, lets you create galleries of images that link to affiliate links (powered by ThirstyAffiliates)

== Description ==

Website: http://thirstyaffiliates.com
PRO Add-ons: http://thirstyaffiliates.com/add-ons

ThirstyAffiliates empowers website owners with the tools they need to monetize their WordPress website with affiliate marketing.

FooGallery is touted as the most intuitive and extensible gallery creation and management tool for WordPress.

Combine them together with this FooGallery extension and you get galleries that can link to affiliate links that are powered by ThirstyAffiliates.

= Features at a glance =

* All the features of [ThirstyAffiliates](http://wordpress.org/plugins/thirstyaffiliates/)
* Adds a new type of gallery link type called Affiliate Links.
* Load affiliate links on images in your gallery individually. One link per gallery image.
* Non-linked gallery images will still use your gallery lightbox.
* Handy affiliate link picker tool (with full instant search capabilities) makes it easy to insert affiliate links on gallery items.
* All the DoFollow/NoFollow, opening in new window, etc options from ThirstyAffiliates are respected
* Great for adding a visually enticing list of affiliate links to your post or page.

= Read more about the required plugins =

You will need both [FooGallery](http://wordpress.org/plugins/foogallery/) and [ThirstyAffiliates](http://wordpress.org/plugins/thirstyaffiliates/) installed.

[Check out all the ThirstyAffiliates Add-ons](http://thirstyaffiliates.com/add-ons)
[Check out all the FooGallery Extensions](http://foo.gallery/)

== Installation ==

1. Upload the `thirstyaffiliates-for-foogallery/` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setup your gallery as normal in FooGallery, choosing the Affiliate Link link type
1. Hover over each gallery item, you can click the familiar green 'aff' icon to load the link you wish for that gallery item
